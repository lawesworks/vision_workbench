Prerequisites

Before starting, acquire free accounts for the services below.

Roboflow Account

Required for retrieving your API key and connecting the notebook in order to download datasets. Visit the API Guide to learn more.

Google Account

Required to open and run Google Colab. Store Roboflow API Key as a Colab Secret called "ROBOFLOW_API_KEY". Visit Secrets Guide

GitHub Account -   Optional

Helpful for accessing shared notebooks, version history, and any repository content associated with the project.
