How to Publish an RTSP Stream ?    ▼

RTSP Stream setup: Server Side

Clear port:
sudo fuser -k 8554/tcp

Start RTSP:
/usr/local/bin/mediamtx /usr/local/etc/mediamtx.yml

Publish to RTSP (local machine)(file):
ffmpeg -re -stream_loop -1 -i input.mp4 -c copy -f rtsp rtsp://localhost:8554/live

Publish to RTSP (file):
ffmpeg -re -stream_loop -1 -i input.mp4 -c copy -f rtsp rtsp://192.168.1.160:8554/live

Publish to RTSP (webcam):
ffmpeg -re -f v4l2 -i /dev/video0 -c:v libx264 -preset veryfast -tune zerolatency -f rtsp rtsp://192.168.1.160:8554/live


RTSP Inference: Client / Inference Side

Consume RTSP feed
http://127.0.0.1:8000/stream/rtsp?url=rtsp://192.168.1.160:8554/live
